# mailing-api
Mailing platform agnostic mailing api for email template application and delivery

## Getting Started
### Installation
```bash
npm install
```

### Running the Server
```bash
npm start
```

## API Routes
| Route | Description |
|-------|-------------|
| `GET /api` | Health check - verify server is running |

**Note:** Additional routes will be added in future releases.

## Features
- Email notification delivery
- Server health monitoring
