# nodemail-server
nodemail server for email notifications/delivery (might be used for porfolio)

## Overview
A Node.js email server for email notifications/message delivery.

## Getting Started
### Installation
```bash
npm install
```

### Running the Server
```bash
npm start
```

## API Routes
| Route | Description |
|-------|-------------|
| `GET /api` | Health check - verify server is running |

**Note:** Additional routes will be added in future releases.

## Features
- Email notification delivery
- Server health monitoring
